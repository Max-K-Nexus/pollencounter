============================================================
  CONTA POLLINICA - ISTRUZIONI PER WINDOWS
============================================================


OPZIONE 1: CREARE L'ESEGUIBILE (CONSIGLIATA)
---------------------------------------------
Questa opzione crea un file .exe che funziona su qualsiasi
PC Windows senza dover installare nulla.

PASSI (da fare UNA SOLA VOLTA su un PC con internet):

  1. Installa Python da: https://www.python.org/downloads/
     IMPORTANTE: durante l'installazione, spunta la casella
     "Add Python to PATH" nella prima schermata.

  2. Copia l'intera cartella "windows" su quel PC.

  3. Fai doppio clic su "build_exe.bat" e attendi.
     Lo script installa le dipendenze e crea l'eseguibile.

  4. Al termine, troverai il file:
       dist\Conta_Pollinica.exe

  5. Copia "Conta_Pollinica.exe" dove vuoi (desktop,
     chiavetta USB, cartella condivisa...).
     Il template Excel e' gia' incluso dentro il .exe.

  6. Per usarlo: doppio clic su Conta_Pollinica.exe.
     Il file di output .xlsx verra' salvato nella stessa
     cartella dove si trova l'eseguibile.


OPZIONE 2: USARE DIRETTAMENTE CON PYTHON
-----------------------------------------
Se il PC ha gia' Python installato:

  1. Installa la dipendenza (una sola volta):
       pip install openpyxl

  2. Fai doppio clic su "AVVIA_CONTA_POLLINICA.bat"
     (installa openpyxl automaticamente se mancante,
     poi apre l'interfaccia grafica).

     In alternativa, da Prompt dei comandi:
       python polline_counter_gui.py

  Nota: i file "polline_counter.py", "polline_counter_gui.py"
  e "Polline_Template_Settimanale.xlsx" devono trovarsi
  nella stessa cartella.


INTERFACCIA GRAFICA
--------------------
La finestra e' divisa in due pannelli:

  PANNELLO SINISTRO - Terminale interattivo
    Mostra l'output dello script e il prompt di inserimento.
    Digita i comandi nel campo in basso e premi INVIO.

  PANNELLO DESTRO - Riepilogo live
    Si aggiorna automaticamente ogni 3 secondi durante la
    sessione, mostrando i conteggi del file Excel corrente.

    Scheda "Settimanale": totale per specie sull'intera
      settimana, con subtotali pollini / spore / TOTALE.

    Scheda "Giornaliero": griglia specie x giorno (lun-dom)
      con subtotali per ciascun giorno.


UTILIZZO DEL PROGRAMMA
-----------------------
All'avvio viene mostrata la lista dei codici disponibili.

  PASSO 1 - Scegliere il file di lavoro
    Se nella cartella esistono file .xlsx precedenti, viene
    proposto di riprenderli oppure di iniziare un file nuovo.

  PASSO 2 - Scegliere la settimana
    Inserisci una data qualsiasi della settimana di lavoro.
    Formati accettati (giorno e mese anche senza zero):
      9-2-2026    09/02/2026    9 feb 2026
    Premi INVIO per usare la settimana corrente.

  PASSO 3 - Scegliere il giorno
    Viene mostrato il calendario della settimana con le date.
    Digita il numero del giorno (1=Lun ... 7=Dom).

  PASSO 4 - Inserire i dati
    Digita il codice specie (01-59) e premi INVIO.
    Il contatore nel prompt si aggiorna ad ogni inserimento.


COMANDI DISPONIBILI
--------------------
  01-59       Inserisce la specie corrispondente
  NNxQ        Inserisce Q occorrenze (es. 48x4 = 4 Alternaria)
  .           Ripete l'ultimo codice inserito
  u           Annulla l'ultimo inserimento
  r           Riepilogo della giornata corrente
  w           Riepilogo settimanale (tutti i giorni)
  l           Ultimi 10 inserimenti (storico)
  c           Correggi un valore di un giorno precedente
  n           Aggiungi una nota testuale alla giornata
  b           Attiva/disattiva beep sonoro ad ogni inserimento
  h           Mostra la tabella dei codici
  d           Chiudi la giornata e passa a un altro giorno
  s           Sospendi sessione (salva e riprendi dopo)
  q           Esci (apre menu salvataggio)


SALVATAGGIO
-----------
  Autosave automatico
    Ogni 10 inserimenti il file viene salvato silenziosamente
    come "~autosave_<data-lunedi>.xlsx". In caso di chiusura
    improvvisa il lavoro e' protetto.

  Comando "s" - Sospensione
    Salva immediatamente come "incompleto_<data>.xlsx" e
    chiude lo script. Alla prossima apertura verra' proposto
    di riprendere da dove si era rimasti.

  Comando "q" - Uscita con menu
    Vengono proposte tre opzioni:
      1) Salvare su un file nuovo (con nome personalizzabile)
      2) Sovrascrivere un file esistente
      3) Uscire senza salvare

  Il template "Polline_Template_Settimanale.xlsx" non viene
  mai modificato.
